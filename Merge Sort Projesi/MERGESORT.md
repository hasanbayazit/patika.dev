# Insertion Sort Projesi
 **[22,27,16,2,18,6]** -> Insertion Sort
## 1. Dizinin sort türüne göre aşamaları
```
[16, 21, 11, 8, 12, 22]
>[16, 21, 11]			>[8, 12, 22]
>[16, 21] [11]			>[8, 12] [22]
>[16] [21] [11]  		>[8] [12] [22]
>[16, 21] [11]  		>[8, 12] [22]
>[11, 16, 21]    		>[8, 12, 22]
>[8, 11, 12, 16, 21, 22 ]
```
## 2. Big-O Gösterimi
```
O(nlogn)
```
